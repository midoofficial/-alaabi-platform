📁 الملفات الأساسية

🔧 إعدادات النشر

  • ملف _redirects للروابط النظيفة
    يحتوي على قواعد إعادة التوجيه للروابط مثل /learning و /activities
  • ملف netlify.toml للإعدادات
    يحتوي على إعدادات الأمان والأداء والتخزين المؤقت
  • ملف manifest.webmanifest للـ PWA
    يجعل الموقع قابل للتثبيت كتطبيق على الهاتف
  • Service Worker (sw.js)
    للعمل بدون إنترنت والتخزين المؤقت الذكي

🔍 تحسينات SEO

  • ملف robots.txt
    يوجه محركات البحث لفهرسة الصفحات المناسبة
  • ملف sitemap.xml
    يساعد محركات البحث في فهرسة جميع صفحات الموقع
  • Meta tags محسّنة
    Open Graph للمشاركة الاجتماعية وTwitter Cards
  • Canonical URLs
    لتجنب المحتوى المكرر في محركات البحث

📊 التحليلات والأداء

  • نظام التحليلات جاهز
    Google Analytics وتتبع الأحداث المخصص
  • مراقبة الأداء
    Core Web Vitals وPerformance monitoring
  • إدارة الخصوصية
    نظام موافقة المستخدم على ملفات تعريف الارتباط

📝 النماذج والتواصل

  • نماذج Netlify Forms
    نماذج التواصل مع التحقق من صحة البيانات
  • حماية من الـ Spam
    Honeypot fields وvalidation للحماية من الرسائل المزعجة
  • صفحة التواصل كاملة
    معلومات التواصل والأسئلة الشائعة

🔒 الأمان

  • رؤوس الأمان
    HTTPS، XSS Protection، Content Security Policy
  • حماية البيانات
    تشفير البيانات وحماية معلومات الأطفال
