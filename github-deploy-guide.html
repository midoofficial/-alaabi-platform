1 تثبيت Git

أ. تحميل Git

اذهب إلى الموقع الرسمي وحمّل Git لنظام Windows:

تحميل Git for Windows

ب. تثبيت Git

  1. شغّل الملف المحمّل
  2. اتبع معالج التثبيت مع الإعدادات الافتراضية
  3. تأكد من تحديد "Git Bash Here" و "Git GUI Here"
  4. اختر "Use Git from the Windows Command Prompt"

ج. التحقق من التثبيت

بعد التثبيت، أعد تشغيل PowerShell وتحقق من التثبيت:

git --version

يجب أن ترى رقم إصدار Git

د. إعداد Git للمرة الأولى

git config --global user.name "اسمك هنا" git config --global user.email "your-email@example.com"
مهم: استخدم نفس البريد الإلكتروني المسجل في GitHub

2 إعداد GitHub

أ. إنشاء حساب GitHub (إذا لم يكن لديك)

إنشاء حساب GitHub

ب. إنشاء مستودع جديد

  1. اذهب إلى github.com
  2. اضغط على زر "New" الأخضر أو "+"
  3. اختر "New repository"
  4. املأ البيانات كما يلي:
Repository name: alaabi-platform Description: منصة ألعابي مع الأخصائية ميرنا السيد - منصة تعليمية عربية للأطفال ☐ Public (أو Private حسب الرغبة) ☐ لا تضع ✅ في Add a README file ☐ لا تضع ✅ في Add .gitignore ☐ لا تضع ✅ في Choose a license

اضغط "Create repository"

ج. نسخ رابط المستودع

بعد إنشاء المستودع، انسخ الرابط الذي يظهر (مثل):

https://github.com/YOUR-USERNAME/alaabi-platform.git

3 رفع المشروع إلى GitHub

✅ الآن: بعد تثبيت Git، نفذ الأوامر التالية في PowerShell

أ. الانتقال لمجلد المشروع

cd "d:\منصة العابي"

ب. إنشاء مستودع Git محلي

git init

ج. إضافة جميع الملفات

git add .

د. إنشاء أول commit

git commit -m "إعداد منصة ألعابي - النسخة الأولى"

هـ. ربط المستودع المحلي بـ GitHub

git remote add origin https://github.com/YOUR-USERNAME/alaabi-platform.git
مهم: استبدل YOUR-USERNAME باسم المستخدم الخاص بك في GitHub

و. تحديد الفرع الرئيسي

git branch -M main

ز. رفع المشروع إلى GitHub

git push -u origin main

قد يطلب منك تسجيل الدخول إلى GitHub

4 ربط GitHub بـ Netlify

أ. الذهاب إلى Netlify

فتح Netlify

سجل دخول أو أنشئ حساب جديد

ب. إنشاء موقع جديد

  1. اضغط "New site from Git"
  2. اختر "GitHub"
  3. سجل دخول إلى GitHub إذا طُلب منك
  4. اختر مستودع "alaabi-platform"

ج. إعدادات النشر

Branch to deploy: main Build command: (اتركه فارغاً) Publish directory: .

اضغط "Deploy site"

د. انتظار اكتمال النشر

سيستغرق النشر 1-3 دقائق. ستحصل على رابط مثل:

https://amazing-site-123456.netlify.app

5 إعدادات ما بعد النشر

🔗 مهم: بعد النشر، ستحتاج لتحديث الروابط في المشروع

أ. تحديث الروابط

استخدم أداة تحديث الروابط:

🔗 أداة تحديث الروابط

ب. إعداد Google Analytics

  1. أنشئ حساب Google Analytics
  2. احصل على Measurement ID
  3. استبدل GA_MEASUREMENT_ID في الملفات

ج. تخصيص اسم الموقع

في Netlify:

  1. Site settings > Domain management
  2. Options > Edit site name
  3. غيّر إلى اسم مناسب مثل: alaabi-platform

6 التحديثات المستقبلية
