2 إعدادات ما بعد النشر

مهم: بعد النشر، ستحتاج لتحديث بعض الإعدادات

تحديث الروابط:

استبدل https://your-site.netlify.app برابط موقعك الفعلي في:

  • ملف index.html (meta tags)
  • ملف robots.txt
  • ملف sitemap.xml

إعداد Google Analytics:

  1. أنشئ حساب على Google Analytics
  2. احصل على Measurement ID
  3. استبدل GA_MEASUREMENT_ID في الملفات

3 اختبار الموقع

تأكد من عمل هذه المميزات:

  • الصفحة الرئيسية تعمل
  • الروابط النظيفة تعمل (/learning, /activities)
  • النماذج ترسل الرسائل
  • الألعاب تعمل بشكل صحيح
  • PWA يعمل (أيقونة التثبيت تظهر)
نصيحة: استخدم أدوات اختبار الأداء مثل PageSpeed Insights

تعليمات Git المفصلة

1. تثبيت Git:

حمّل وثبّت Git من الموقع الرسمي

2. إعداد Git:

git config --global user.name "اسمك"
git config --global user.email "your-email@example.com"

3. إنشاء مستودع:

cd "d:\منصة العابي"
git init
git add .
git commit -m "إعداد منصة ألعابي"

4. ربط بـ GitHub:

  1. أنشئ مستودع جديد على GitHub
  2. انسخ رابط المستودع
  3. نفذ الأوامر:
git remote add origin https://github.com/USERNAME/REPO-NAME.git
git branch -M main
git push -u origin main

5. ربط بـ Netlify:

  1. في Netlify، اضغط "New site from Git"
  2. اختر GitHub واختر المستودع
  3. اضغط Deploy
